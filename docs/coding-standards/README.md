# Coding Standards

> This is the **single source of truth** for all coding standards across Weblabor projects.
> All AI tools (Claude, Copilot), agents (analyst, developer, code-reviewer, tech-lead, frontend, ux-designer, workflow-reviewer), and developers must follow these rules.
>
> When user feedback corrects or adds a standard, propose the relevant file update and apply it only after explicit confirmation. Changes to standards, commands, agents, or workflow documentation must be reviewed by `workflow-reviewer` before they are treated as final.
